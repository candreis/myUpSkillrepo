Descrição do curso

Programação Web

HTML CSS JavaScript

Objetivos e Conteúdos

Objetivos de aprendizagem

  • Elaborar projetos web, aplicando metodologia/Processo de desenvolvimento de software, desde a fase de análise de requisitos, até a publicação da aplicação web.
  • Programar aplicações web usando HTML5, CSS3 e JavaScript 5 com armazenamento local, processamento assíncrono e redirecionamento no servidor.
  • Avaliar a importância das frameworks e APIs JavaScript e CSS no desenvolvimento de aplicações web.
